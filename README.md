# Hiree - Freelancing Platform

A full-stack freelancing platform built with MERN stack.

## 📁 Project Structure

- **`/backend`** - Node.js + Express API
- **`/frontend`** - React + Vite frontend

## 🚀 Tech Stack

- **Frontend:** React, Vite, Bootstrap
- **Backend:** Node.js, Express, MongoDB
- **Real-time:** Socket.IO
